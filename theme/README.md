# ko-theme
> dtux component ko theme.


## Installation
```
yarn add ko-theme  --save
```

## Usage

Use Sass import
```css
@import 'ko-theme';
```

Or Use webpack
```javascript
import 'ko-theme';
```

Or
```html
<link rel="stylesheet" href="node_modules/ko-theme/lib/index.css">
```

##  Import on demand
```javascript
import 'ko-theme/lib/input.css';
import 'ko-theme/lib/select.css';

// ...
```
